# Polygon
# Full Stack CRUD App

## Tech Stack
- **Frontend**: Next.js 14, TypeScript, MUI, localStorage
- **Backend**: Node.js, Express, MySQL, JWT, Zod

## Features
- [x] Task CRUD with form validation
- [x] Responsive MUI UI
- [x] User register/login/update
- [x] Secure JWT auth
- [x] Post CRUD per user
- [x] SQL schema
- [x] Proper HTTP codes & error handling
- [x] Email validation
